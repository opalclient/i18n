# Player & World

---

## `player` — PlayerProxy

Exposes local player state, position, health, combat utilities, and actions.

### Position

| Method | Returns | Description |
|---|---|---|
| `getBlockPosition()` | `BlockPos` | Floored block coordinates the player is standing in |
| `getPosition()` | `Vec3d` | Eye position in world space |
| `getEyePosition()` | `Vec3d` | Eye position in world space |
| `getVelocity()` | `Vec3d` | Current velocity vector |
| `getBoundingBox()` | `Box` | Player's axis-aligned bounding box |
| `getStandingEyeHeight()` | `float` | Eye height above feet |

### Rotation

| Method | Returns | Description |
|---|---|---|
| `getYaw()` | `float` | Current yaw in degrees |
| `getPitch()` | `float` | Current pitch in degrees |

### State

| Method | Returns | Description |
|---|---|---|
| `isOnGround()` | `boolean` | Whether the player is on the ground |
| `isInAir()` | `boolean` | Whether the player is airborne |
| `getAirTicks()` | `int` | Ticks spent in the air continuously |
| `getGroundTicks()` | `int` | Ticks spent on the ground continuously |
| `isSneaking()` | `boolean` | Whether the player is sneaking |
| `isSprinting()` | `boolean` | Whether the player is sprinting |
| `isUsingItem()` | `boolean` | Whether the player is eating, blocking, drawing a bow, etc. |
| `getFallDistance()` | `double` | Current fall distance in blocks |

### Health

| Method | Returns | Description |
|---|---|---|
| `getHealth()` | `float` | Current health points |
| `getMaxHealth()` | `float` | Maximum health points |
| `getAbsorption()` | `float` | Absorption (golden hearts) amount |

### Combat

| Method | Returns | Description |
|---|---|---|
| `canCrit()` | `boolean` | Whether the next attack will be a critical hit |
| `getAttackDamage()` | `double` | Main-hand attack damage |
| `getEntityInteractionRange()` | `double` | Maximum entity attack range |
| `isHoldingWeapon()` | `boolean` | Whether the player holds a sword, axe, or pickaxe |
| `getDistanceToEntity(entity)` | `double` | Box-to-box distance to the given entity |
| `getClosestPoint(entity)` | `Vec3d` | Closest bounding-box point on an entity from the player's eye |

### Bounding box checks

| Method | Returns | Description |
|---|---|---|
| `isBoxEmpty(dx, dy, dz)` | `boolean` | Whether an offset of the player's box has no collision |
| `isBoxEmptyBelow(dy)` | `boolean` | Whether the space below the player (at `dy`) is empty |

### Actions

| Method | Returns | Description |
|---|---|---|
| `swingHand(hand)` | `void` | Plays the hand swing animation. Use `MAIN_HAND` or `OFF_HAND`. |

```js
player.swingHand(MAIN_HAND);
```

## `world` — WorldProxy

Provides block queries, adjacent direction lookups, and entity searches.

### Block queries

| Method | Returns | Description |
|---|---|---|
| `isReplaceable(blockPos)` | `boolean` | Whether the block at the given position can be replaced (air, fluid, grass, etc.) |
| `isSolid(blockPos)` | `boolean` | Whether the block at the given position is solid |
| `hasAdjacentBlock(blockPos)` | `boolean` | Whether any face-adjacent block is solid and placeable-against |
| `getAdjacentDirections(blockPos)` | `List<Direction>` | List of directions where a solid neighbour exists (can be empty) |

```js
const below = new BlockPos(pos.getX(), pos.getY() - 1, pos.getZ());

if (!world.isReplaceable(below)) return;

const dirs = world.getAdjacentDirections(below);
if (dirs.isEmpty()) return;

const side = dirs.get(0);
const neighbor = below.offset(side);
```

### Entity queries

| Method | Returns | Description |
|---|---|---|
| `getNearbyEntities(range)` | `List<Entity>` | All entities within `range` blocks of the player |
| `getNearbyPlayers(range)` | `List<PlayerEntity>` | Nearby players only, excluding self |
| `getNearbyLivingEntities(range)` | `List<LivingEntity>` | Nearby living entities (mobs + players), excluding self |

## ESP Proxy — `esp`

Provides 3D-to-2D screen projection for custom ESP overlays.

| Method | Returns | Description |
|---|---|---|
| `project(x, y, z)` | `double[2]` or `null` | Screen `[x, y]` for the given world position, or `null` if behind the camera |
| `getEntityScreenPos(entity)` | `double[2]` or `null` | Screen position of an entity's origin |
| `isOnScreen(x, y, z)` | `boolean` | Whether a world position projects within the visible viewport |

```js
const pos = esp.getEntityScreenPos(target);
if (pos !== null) {
    renderer.text("productsans-bold", "target", pos[0], pos[1], 8, 0xFFFFFFFF);
}
```
