<Method name="getAdjacentDirections" signature="List&lt;Direction&gt; getAdjacentDirections(BlockPos pos)" description="Returns directions where a solid neighbour exists.
```javascript
const below = new BlockPos(pos.getX(), pos.getY() - 1, pos.getZ());

if (!world.isReplaceable(below)) return;

const dirs = world.getAdjacentDirections(below);
if (dirs.isEmpty()) return;

const side = dirs.get(0);
const neighbor = below.offset(side);
```

### Example — Custom Name ESP

```javascript
module.on("renderScreen", (event) => {
    if (mc.player === null || mc.world === null) return;
    const tickDelta = client.getTickDelta();

    const entities = world.getLivingEntitiesInRange(64);
    for (const entity of entities) {
        const box = esp.getEntityBox2D(entity, tickDelta);
        if (box === null) continue;

        // Bounding box outline
        renderer.rectOutline(box.x, box.y, box.z, box.w, 1.0, client.getThemePrimary());

        // Name tag above box
        const name = entity.getName().getString();
        const tw = renderer.textWidth("productsans-medium", name, 7);
        const tagX = box.x + (box.z / 2) - (tw / 2);
        const tagY = box.y - 14;

        renderer.roundedRect(tagX - 4, tagY - 3, tw + 8, 14, 3, renderer.color(0, 0, 0, 160));
        renderer.text("productsans-medium", name, tagX, tagY + 7, 7, 0xFFFFFFFF);
    }
});
```

## Types

Script-safe wrapper types for Minecraft objects. These provide stable, non-intermediary method names.

### BlockPos

Represents an integer block position. Construct with `new BlockPos(x, y, z)`.

```javascript
const pos = player.getBlockPosition();
const below = new BlockPos(pos.getX(), pos.getY() - 1, pos.getZ());
```

| Method | Returns | Description |
|---|---|---|
| `getX()` | `int` | X coordinate |
| `getY()` | `int` | Y coordinate |
| `getZ()` | `int` | Z coordinate |
| `offset(direction)` | `BlockPos` | New BlockPos shifted one block in the given Direction |

### Vec2f

Represents a yaw/pitch pair. Construct with `new Vec2f(yaw, pitch)`.

```javascript
const rot = rotation.getRotation();
client.print("Yaw: " + rot.getYaw() + " Pitch: " + rot.getPitch());
```

| Method | Returns | Description |
|---|---|---|
| `getYaw()` | `float` | Yaw in degrees |
| `getPitch()` | `float` | Pitch in degrees |

### Direction

Represents a cardinal or vertical facing direction. Comes from `world.getAdjacentDirections()`.

| Method | Returns | Description |
|---|---|---|
| `getOpposite()` | `Direction` | The opposite direction |
| `getName()` | `String` | Lowercase name: `"north"`, `"south"`, `"east"`, `"west"`, `"up"`, `"down"` |

### RaytracedRotation

Returned by `rotation.getRotationFromRaycastedBlock()` and `rotation.getRotationFromRaycastedEntity()`. Contains the validated rotation and the hit result needed for block interaction.

```javascript
const raytrace = rotation.getRotationFromRaycastedBlock(
    neighbor, side.getOpposite(), rotation.getRotation(), player.getEyePosition()
);
if (raytrace === null) return;

rotation.set(raytrace.getYaw(), raytrace.getPitch());
mc.interactionManager.interactBlock(MAIN_HAND, raytrace.getHitResult());
```

| Method | Returns | Description |
|---|---|---|
| `getYaw()` | `float` | Target yaw |
| `getPitch()` | `float` | Target pitch |
| `getHitResult()` | `HitResult` | Pass to `mc.interactionManager.interactBlock()` |

### Hand Constants

| Constant | Description |
|---|---|
| `MAIN_HAND` | Player's main hand |
| `OFF_HAND` | Player's off hand |
