# Core Proxies

These proxies handle general client interaction, notifications, and UI.

---

## Client Proxy
**Global Binding:** `client`

Provides access to high-level client functions, including module management, chat interaction, and system metrics.

### Chat & Messaging

<Method
    name="print"
    signature="void print(Object o)"
    description="Prints a message to the in-game chat for the local player only. Calls toString() on the object."
    params={[
        { name: "o", type: "Object", desc: "The object or message to print." }
    ]}
/>

<Method
    name="success"
    signature="void success(String message)"
    description="Prints a success-styled (green/checkmarked) message to the local chat."
    params={[
        { name: "message", type: "String", desc: "The success message text." }
    ]}
/>

<Method
    name="error"
    signature="void error(String message)"
    description="Prints an error-styled (red/crossed) message to the local chat."
    params={[
        { name: "message", type: "String", desc: "The error message text." }
    ]}
/>

## Overlay Proxy
**Global Binding:** `overlay`

Allows scripts to register and manage custom "Dynamic Island" triggers on the HUD.

### Creation & Management

<Method
    name="createIsland"
    signature="String createIsland(Object config)"
    description="Creates a new island and returns its unique ID."
    params={[
        { name: "config", type: "Object", desc: "Configuration object: { width, height, priority, render(x,y,w,h,p) }." }
    ]}
    returns={{ type: "String", desc: "Unique Island ID." }}
/>

<Method
    name="showIsland"
    signature="void showIsland(String islandId)"
    description="Activates the island, adding it to the render loop."
    params={[
        { name: "islandId", type: "String", desc: "The ID returned by createIsland." }
    ]}
/>

<Method
    name="hideIsland"
    signature="void hideIsland(String islandId)"
    description="Deactivates the island, removing it from the render loop."
    params={[
        { name: "islandId", type: "String", desc: "The ID returned by createIsland." }
    ]}
/>

<Method
    name="destroyIsland"
    signature="void destroyIsland(String islandId)"
    description="Permanently deletes the island trigger."
    params={[
        { name: "islandId", type: "String", desc: "The ID returned by createIsland." }
    ]}
/>

### Updates

<Method
    name="setIslandWidth"
    signature="void setIslandWidth(String islandId, float width)"
    description="Updates the content width of the island dynamically."
    params={[
        { name: "islandId", type: "String", desc: "The ID returned by createIsland." },
        { name: "width", type: "float", desc: "New width in pixels." }
    ]}
/>

<Method
    name="setIslandHeight"
    signature="void setIslandHeight(String islandId, float height)"
    description="Updates the height of the island."
    params={[
        { name: "islandId", type: "String", desc: "The ID returned by createIsland." },
        { name: "height", type: "float", desc: "New height in pixels." }
    ]}
/>

<Method
    name="setIslandPriority"
    signature="void setIslandPriority(String islandId, int priority)"
    description="Updates the render priority."
    params={[
        { name: "islandId", type: "String", desc: "The ID returned by createIsland." },
        { name: "priority", type: "int", desc: "New priority (higher renders on top)." }

/>
