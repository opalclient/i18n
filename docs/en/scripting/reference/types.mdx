# Types

These types are exposed as globals in every script. They wrap underlying Minecraft objects to provide stable, non-intermediary method names that work correctly at runtime under Fabric mappings.

---

## BlockPos

Represents an integer block position in the world. Bound as `BlockPos` so you can construct one with `new BlockPos(x, y, z)`.

```js
const pos = player.getBlockPosition(); // returns BlockPos
const below = new BlockPos(pos.getX(), pos.getY() - 1, pos.getZ());
```

| Method | Returns | Description |
|---|---|---|
| `getX()` | `int` | X coordinate |
| `getY()` | `int` | Y coordinate |
| `getZ()` | `int` | Z coordinate |
| `offset(direction)` | `BlockPos` | Returns a new `BlockPos` shifted one block in the given `Direction` |

---

## Vec2f

Represents a 2D float vector (yaw/pitch pair). Bound as `Vec2f` so you can construct one with `new Vec2f(yaw, pitch)`.

```js
const rot = rotation.getRotation(); // returns Vec2f
client.print("Yaw: " + rot.getYaw() + "  Pitch: " + rot.getPitch());
```

| Method | Returns | Description |
|---|---|---|
| `getYaw()` | `float` | The X component (yaw in degrees) |
| `getPitch()` | `float` | The Y component (pitch in degrees) |

---

## Direction

Represents a cardinal or vertical facing direction. You never construct one directly â€” they come from `world.getAdjacentDirections()`.

```js
const dirs = world.getAdjacentDirections(below); // returns List<Direction>
const side = dirs.get(0);
const opp  = side.getOpposite();
```

| Method | Returns | Description |
|---|---|---|
| `getOpposite()` | `Direction` | The direction directly opposite this one |
| `getName()` | `String` | Lowercase name, e.g. `"north"`, `"up"`, `"down"` |

## ScriptRaytracedRotation

Returned by `rotation.getRotationFromRaycastedBlock()` and `rotation.getRotationFromRaycastedEntity()`. Contains the yaw/pitch to aim at the target **and** the hit result needed to place/interact with the block.

```js
const raytrace = rotation.getRotationFromRaycastedBlock(
    neighbor,
    side.getOpposite(),
    rotation.getRotation(),
    player.getEyePosition()
);
if (raytrace === null) return;

rotation.set(raytrace.getYaw(), raytrace.getPitch());
mc.interactionManager.interactBlock(MAIN_HAND, raytrace.getHitResult());
```

| Method | Returns | Description |
|---|---|---|
| `getYaw()` | `float` | Yaw angle in degrees to look at the target |
| `getPitch()` | `float` | Pitch angle in degrees to look at the target |
| `getHitResult()` | `HitResult` | Raw hit result to pass to `mc.interactionManager.interactBlock()` |
