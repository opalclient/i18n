# Module Proxy

**Global Binding:** `modules`

Provides dedicated access to the client's module system. Allows scripts to query module state, toggle modules, and enumerate modules by category.

<Callout type="info">
    All module lookups are case-insensitive. Methods return safe defaults (false, null, or empty arrays) if the requested module does not exist.
</Callout>

---

## State Queries

<Method
    name="exists"
    signature="boolean exists(String id)"
    description="Checks whether a module with the given name is registered."
    params={[
        { name: "id", type: "String", desc: "The module display name." }
    ]}
    returns={{ type: "boolean", desc: "True if the module exists." }}
/>

<Method
    name="isEnabled"
    signature="boolean isEnabled(String id)"
    description="Checks if a specific module is currently enabled."
    params={[
        { name: "id", type: "String", desc: "The module display name." }
    ]}
    returns={{ type: "boolean", desc: "True if enabled, false if disabled or not found." }}
/>

<Method
    name="setEnabled"
    signature="void setEnabled(String id, boolean enabled)"
    description="Enables or disables a module by name. No-ops if the module does not exist."
    params={[
        { name: "id", type: "String", desc: "The module display name." },
        { name: "enabled", type: "boolean", desc: "True to enable, false to disable." }
    ]}
/>

<Method
    name="toggle"
    signature="void toggle(String id)"
    description="Toggles a module on or off. Triggers the same notification as a keybind toggle. No-ops if the module does not exist."
    params={[
        { name: "id", type: "String", desc: "The module display name." }
    ]}
/>

## Module Metadata

<Method
    name="getCategory"
    signature="String getCategory(String id)"
    description="Returns the category name of a module (e.g. 'Combat', 'Movement', 'Visual')."
    params={[
        { name: "id", type: "String", desc: "The module display name." }
    ]}

/>

<Method
    name="getSuffix"
    signature="String getSuffix(String id)"

        { name: "id", type: "String", desc: "The module display name." }
    ]}
    returns={{ type: "String", desc: "Suffix string, or null if the module has no suffix." }}
/>

<Method

    params={[
        { name: "id", type: "String", desc: "The module display name." }
    ]}
    returns={{ type: "boolean", desc: "True if visible." }}
/>

<Method
    name="setVisible"

## Listing

/>

<Method
