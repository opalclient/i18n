# Renderer Proxy

**Global Binding:** `renderer`

The renderer provides a complete 2D drawing API backed by **NanoVG**. All drawing calls must be made inside a `renderScreen` event callback, where the NVG frame is active.

<Callout type="info">
    All color parameters expect a packed **ARGB Integer**. Create these using `renderer.color(r, g, b, a)` or the `Color` Java class.
</Callout>

## Path API

For custom vector shapes. Begin a path, add segments, then stroke it.

<Method name="beginPath" signature="void beginPath()" description="Starts a new path." />
<Method name="moveTo" signature="void moveTo(double x, double y)" description="Moves the cursor without drawing." params={[{ name: "x", type: "double", desc: "X." }, { name: "y", type: "double", desc: "Y." }]} />
<Method name="lineTo" signature="void lineTo(double x, double y)" description="Draws a line from cursor to target." params={[{ name: "x", type: "double", desc: "X." }, { name: "y", type: "double", desc: "Y." }]} />
<Method name="closePath" signature="void closePath()" description="Closes the path by connecting last point to start." />
<Method name="strokeColor" signature="void strokeColor(double color)" description="Sets stroke color." params={[{ name: "color", type: "double", desc: "ARGB color." }]} />
<Method name="strokeWidth" signature="void strokeWidth(double width)" description="Sets stroke thickness." params={[{ name: "width", type: "double", desc: "Line thickness." }]} />
<Method name="stroke" signature="void stroke()" description="Renders the current path as an outline." />

```javascript
// Draw a triangle
renderer.beginPath();
renderer.moveTo(100, 100);
renderer.lineTo(150, 50);
renderer.lineTo(200, 100);
renderer.closePath();
renderer.strokeColor(0xFFFFFFFF);
renderer.strokeWidth(2);
renderer.stroke();
```

## Full Example

```javascript
module.on("renderScreen", (event) => {
    const sw = client.getScaledWidth();
    const panelWidth = 140;
    const x = sw - panelWidth - 10;
    const y = 10;

    // Background
    renderer.roundedRect(x, y, panelWidth, 60, 6, renderer.color(15, 15, 20, 200));

    // Title with theme color
    renderer.textShadow("productsans-bold", "Status", x + 10, y + 16, 9, client.getThemePrimary());

    // Stats
    renderer.text("productsans-medium", "FPS: " + client.getFPS(), x + 10, y + 34, 7, renderer.color(180, 180, 180));
    renderer.text("productsans-medium", "BPS: " + movement.getBlocksPerSecond().toFixed(1), x + 10, y + 48, 7, renderer.color(180, 180, 180));
});
```
