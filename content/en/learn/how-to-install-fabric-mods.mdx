## What is Fabric?

Fabric is a lightweight mod loader for Minecraft. It's fast, actively maintained, and supports modern Minecraft versions. Many utility clients — including Opal — run on Fabric.

## Prerequisites

Before you start, make sure you have:

- **Minecraft Java Edition** installed and launched at least once
- **Java 21** or newer installed (required for MC 1.21+)
- A working internet connection

## Step 1: Download Fabric Loader

1. Go to the official Fabric website at **fabricmc.net**
2. Click **Download** and select the **Installer** option
3. Download the Universal JAR installer

## Step 2: Install Fabric

1. Run the Fabric installer JAR you downloaded
2. Make sure these settings are correct:
   - **Client** tab is selected
   - **Minecraft Version:** 1.21.10 (or your target version)
   - **Loader Version:** Latest stable (0.17.x recommended)
   - **Install Location:** Your default `.minecraft` directory
3. Click **Install**
4. You should see a success message

## Step 3: Verify the Installation

1. Open the Minecraft Launcher
2. Click the **Installations** tab
3. You should see a new profile called **fabric-loader-1.21.10** (or similar)
4. Select it and click **Play**
5. If Minecraft launches successfully with the Fabric logo on the title screen, you're good

## Step 4: Install Mods

Fabric mods go in your `mods` folder:

- **Windows:** `%appdata%\.minecraft\mods`
- **macOS:** `~/Library/Application Support/minecraft/mods`
- **Linux:** `~/.minecraft/mods`

If the `mods` folder doesn't exist, create it.

To install a mod:
1. Download the mod's `.jar` file
2. Place it in the `mods` folder
3. Launch Minecraft with the Fabric profile

## Installing Opal Client

Opal uses a **loader system** — you don't drop the full client into the mods folder. Instead:

1. Purchase Opal from the dashboard ($8.99 lifetime)
2. Download the **Opal Launcher** from the downloads page
3. The launcher handles authentication, installation, and updates automatically
4. Alternatively, download the Opal Loader JAR and place it in your `mods` folder — it will authenticate and download the client at runtime

The launcher is the recommended approach — it manages everything for you.

## Troubleshooting

### "Fabric doesn't show up in the launcher"
Make sure you installed Fabric for the correct Minecraft version. Re-run the installer if needed.

### "Game crashes on launch"
- Check that you have **Java 21** installed (not Java 8 or 17)
- Remove any incompatible mods from the `mods` folder
- Make sure Fabric Loader is up to date

### "Mods aren't loading"
- Verify the `.jar` files are in the correct `mods` folder
- Make sure you're launching with the **Fabric profile**, not vanilla
- Check that the mod version matches your Minecraft version

### "Opal won't authenticate"
- Make sure you have an active internet connection
- Verify your license key is activated on the dashboard
- Check that your HWID hasn't changed (reset available every 30 days on the dashboard)
- Users in regions with internet restrictions may need a proxy — check the purchase page for details

## Frequently Asked Questions

**Do I need Fabric to use Opal?**
Yes — Opal is a Fabric mod. The Opal Launcher handles Fabric installation automatically, but if you're installing manually, you need Fabric Loader installed first.

**Can I use Fabric mods with Forge?**
No. Fabric and Forge are separate mod loaders with incompatible mod formats. Opal requires Fabric specifically.

**What Java version do I need for Minecraft 1.21?**
Java 21 or newer. Older Java versions (8, 17) won't work with modern Minecraft. Download Java 21 from Adoptium or Oracle.

**Will Fabric mods break my Minecraft installation?**
No. Fabric mods are isolated to the `mods` folder. If something goes wrong, just remove the mod's JAR file and launch vanilla. Your worlds and saves are unaffected.

**Can I run Opal alongside other Fabric mods?**
Generally yes, as long as the other mods don't conflict. Opal is designed to work as a standalone client mod on Fabric 1.21.10.
