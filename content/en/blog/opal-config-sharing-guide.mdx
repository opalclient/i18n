## What Are Configs?

A config in Opal is a saved snapshot of all your module settings — which modules are enabled, their values, keybinds, and parameters. Think of it as a loadout that you can save, share, and swap between.

## Browsing Public Configs

The **config browser** on the Opal dashboard lets you discover configs shared by the community.

### Finding Configs

1. Go to your dashboard and navigate to **Configs**
2. Browse the public config list
3. Use the search bar to find configs by name
4. Sort by: **Newest**, **Most Downloaded**, or **Most Liked**

### What to Look For

Good configs usually have:
- A descriptive name (e.g., "Competitive Bedwars Ghost" or "Blatant Speed Bridge")
- High download and like counts
- Recent upload dates (newer = more likely to work with current bypasses)

## Downloading Configs

Found a config you like? Click on it to view the details, then download it. The config is saved to your private collection and can be loaded in-game.

### Loading In-Game

Use the config command in-game:
```
/load config-name
```

Or browse your saved configs through the in-game **Config Browser** module.

## Creating Your Own Configs

### Step 1: Tune Your Setup

Set up your modules exactly how you want them. Adjust every setting — sensitivity, ranges, delays, visuals.

### Step 2: Save Locally

Save your current setup as a named config. This creates a private config that only you can access.

### Step 3: Publish (Optional)

If you want to share your config with the community:

1. Go to **Configs** on the dashboard
2. Find your saved config
3. Click **Publish** to make it public
4. Add a descriptive name so others can find it

Published configs appear in the public browser for anyone to download.

## Config Strategies

### Server-Specific Configs

Create configs tailored to specific servers or game modes:

- **"Server Ghost"** — subtle aim assist, reach 3.1, velocity 85%
- **"Server Blatant"** — full bypass suite for casual games
- **"Practice PvP"** — ghost features only, tuned for duels

### Playstyle Configs

Match your config to how you play:

- **Aggressive** — higher reach, stronger velocity reduction, faster attack speed
- **Defensive** — focus on velocity and movement, minimal aim assist
- **Balanced** — moderate settings across the board

### Quick-Switch Setup

Use Opal's scripting engine to auto-load configs based on the server you join. This way you always have the right setup without manual switching.

## Tips for Publishing Good Configs

1. **Name it clearly** — "BW_ghost_v3" is better than "my config"
2. **Test before publishing** — make sure everything works together
3. **Update regularly** — if bypasses change, update your config
4. **Include context** — describe what the config is for in the name

## Why Config Sharing Matters

No other client has a built-in config sharing platform. With Rise or Vape, you're copying JSON files through Discord or figuring out settings from YouTube videos.

Opal's config system means:
- **New users** can get competitive immediately by downloading proven configs
- **Experienced users** can share their expertise with the community
- **Everyone** benefits from community-tested, optimized settings

It's one of the features that makes Opal more than just a client — it's a platform.
